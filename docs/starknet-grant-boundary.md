# Starknet Grant Boundary

Spark is not only an education app. It is a Starknet readiness gateway.

Before grant:

- public app
- Core/Lab leveling
- Passport proof foundation
- proof signals
- backend foundation

After backend:

- signed readiness event ledger
- issued Passport records
- media-backed evidence bundles

After grant accepted:

- Cairo PassportRegistry
- Scarb workflow
- Starknet Foundry testing/deployments
- Starknet Sepolia testing
- Starknet Mainnet anchoring
- NFT or non-transferable Passport badge
- public verifier
- community verifier
- StarknetKit wallet/account abstraction readiness
- Hub ecosystem integrations
- optional Dojo/provable Lab sandbox

This boundary prevents overbuilding before resources are available while keeping the grant scope broad and strong.
